## 📌 Sobre o projeto

Este projeto consiste em uma página web que reúne diferentes sites favoritos organizados por categorias.

A interface foi desenvolvida com inspiração retrô, utilizando o framework NES.css, com foco em simplicidade, organização e estética visual.

---

## 🎯 Objetivo

Aplicar os conhecimentos adquiridos em HTML e CSS na criação de uma página web funcional, contendo:

* Estrutura básica de um site
* Uso de imagens (ícones)
* Links externos
* Organização por seções
* Estilização com CSS
